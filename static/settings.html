Anthropic API Key

Not configured

SocialClipper uses Claude AI to analyze your video transcript and write social media drafts. The API key is only used for this processing. Transcription and video editing happen locally on your machine. No video or audio data is ever sent to the API.

How to get an API key:
  1. Go to console.anthropic.com
  2. Sign up or log in
  3. Go to API Keys and create a new key
  4. Copy the key and paste it above
Your key is stored locally on this machine and never shared.

Video Length What's Sent Typical Cost
